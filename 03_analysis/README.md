# Analysis

Use this folder for computational and analytical work.

- Keep original datasets in `01_sources/datasets/`.
- Use `data_raw/` only for local working copies needed by scripts or notebooks.
- Put cleaned or transformed data in `data_processed/`.
- Put reusable code in `scripts/`.
- Put exploratory notebooks in `notebooks/`.
- Put generated figures, tables, reports, and model outputs in `outputs/`.

Analysis outputs should be reproducible from documented inputs whenever possible.
