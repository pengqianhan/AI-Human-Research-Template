# AI-Human Research Template

This repository is a reusable workspace for research projects that involve both human
researchers and AI agents such as Codex or Claude Code.

Start a new project by cloning this template, filling in
`00_project/research_brief.md`, and placing source materials in `01_sources/`.
All operating instructions live in `instruction.md`; both `AGENTS.md` and `CLAUDE.md`
point agents there first.

## Quick Start

1. Clone this template into a new research project folder.
2. Fill in `00_project/research_brief.md`.
3. Add papers, datasets, web captures, or transcripts under `01_sources/`.
4. Use `02_literature/` for reading notes and synthesis.
5. Use `03_analysis/` for code, notebooks, processed data, and outputs.
6. Use `04_writing/` for outlines, evidence mapping, and drafts.
7. Store final manuscripts, slides, and submission files in `06_deliverables/`.

Read `instruction.md` before starting substantive work.
